Predicting Pitchfork Album Ratings from Review Text
Built a regression model by fine-tuning a pre-trained transformer model to predict album ratings based on review text body. The project consists of a sequence of jupyter notebooks and a corresponding custom library. A detailed description of the project can be found in the README document via the link below. The notebooks are available as both the original .ipynb source and .html exports.
